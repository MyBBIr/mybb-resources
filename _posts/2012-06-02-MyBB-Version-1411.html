MyBB 1.4.11 was released on 29 December 2009.

MyBB 1.4.11 has just been released and is available from the MyBB website. This is a minor patch and security release which fixes one security issue, a performance issue, and some regressions reported from MyBB 1.4.10.

Changes for MyBB 1.4.11 include:

  • A low security fix that can allow a user to check for file existence outside of the web root.
  • A performance bug fix that with malicious intent can be used to assist a Denial-of-Service attack
  • Several bug fixes

More information on the MyBB Blog: http://blog.mybb.com/2009/12/29/mybb-1-4-11-released-minor-patch-security-update/
