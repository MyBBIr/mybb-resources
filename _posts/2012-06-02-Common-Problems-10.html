MySQL Error 1136 when importing theme

A MySQL error 1136 occurs when importing a theme. http://community.mybb.com/showthread.php?tid=5239

Removing Administrator Debug Info

At the bottom of the forum, administrators see debug information such as SQL queries, GZip, and generation time. The link below describes how to remove it. http://community.mybb.com/showthread.php?tid=5540

Edit Post Time Restriction Setting

The setting for restricting post editing to a certain time was not included in versions before 1.2.0 by default. http://community.mybb.com/showthread.php?tid=5630

Moving and deleting forums

If you move a subforum to a new parent forum and delete its original parent forum, the subforum may be deleted as well. http://community.mybb.com/showthread.php?tid=6392&pid=39180#pid39180

Login/Logout/Cookie Problems

If you are having login/logout or otherwise suspect cookie problems, see Cookie Settings. Additionally, try to clear your cookies.
