The MyBB Merge System allows for easy merging of an existing forum (be it MyBB or another forum software) into a MyBB 1.6.x forum. The Merge System automatically converts the data from the other forum software (if applicable) into a MyBB-friendly format.

The Merge System only imports into MyBB 1.6.x. The version available on the Merge System downloads page is for MyBB 1.6. The version of the Merge System for importing into MyBB 1.4 is no longer supported.

The following forum software are supported by the MyBB Merge System (1.6.0):

  • bbPress
  • Invision Power Board 2.1, 2.2, or 2.3
  • Invision Power Board 3 - Forthcoming (Beta Version in the SVN Trunk)
  • MyBB 1.6 (Merge)
  • phpBB 2
  • phpBB 3
  • punBB 1.2
  • SMF 1.1, 2.0
  • XMB 1.9 As of the MyBB Merge System 1.6 this module has been dropped due to lack of demand.
  • vBulletin 3.6, 3.7, 3.8
  • vBulletin 4 - Forthcoming (Beta Version in the SVN Trunk)
  • XenForo - Forthcoming

If the forum software you use is not listed, please request it at the MyBB Community Forums.

The MyBB Merge System uses "modules" to take you through the conversion process. This makes it easier for you to choose what you want converted over, and what you don't want converted. You will also notice that some modules have "dependencies," which means it requires the displayed modules to be run before it can run itself.

The modules that are included for each forum software vary based on what that forum supports in comparison to MyBB. For instance, not all software have a "calendar" feature, so only those that do have a module that transfers over the calendar events.

See Running the Merge System.
