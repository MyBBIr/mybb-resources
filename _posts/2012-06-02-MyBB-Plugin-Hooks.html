  • forumdisplay_thread - called for every thread displayed in a forum's list of threads
  • build_forumbits_forum called - for every forum displayed in a list of forums
  • member_do_register_start - called by member.php before creating a new user. If your plugin creates new users (say, an OpenID signup plugin), make sure to run this hook: $plugins->run_hooks("member_do_register_start");.
  • member_do_register_end - called by member.php after a new user has been created. Data available in the $userinfo array. Used by plugins such as Email about new members. If your plugin creates new users (say, a Facebook connect plugin), make sure to run this hook: $plugins->run_hooks("member_do_register_end");.
