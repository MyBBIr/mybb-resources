Welcome to the MyBB General Support portal

This page provides useful information to help you solve problems you might be having with your MyBB installation and to gain support from the community if you require.

The MyBB community forums are the only place to receive official support from the MyBB team. You should post in the General support forums if you are having problems installing, upgrading or using default features within MyBB.

  • MyBB 1.6 General Support Forum

Please note that you should not post in the general support forums if your problem relates to any of the following:

  • Installing or configuring a plugin - Correct forum
  • Installing or modifying a theme - Correct forum
  • Requesting a plugin - Correct forum

You must also meet the Support eligibility requirements to recieve support on the community forums.

Before making a support post please read the MyBB support readme page

Also, please remember to include the following details with your post:

  • Detailed description of your problem, including steps to reproduce if necessary
  • URL to your forum/URL to specific problematic page
  • New installation or upgrade (from which version of MyBB)?
  • Test user account (if a posting/UserCP issue)
  • Screenshot or error text, verbatim
