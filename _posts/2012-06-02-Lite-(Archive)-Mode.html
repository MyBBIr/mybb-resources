Lite (Archive) mode is a search-engine friendly mode that allows search engines and screen readers to easily read the forum. The Archive mode only allows 'reading,' and so one must access the 'full' version of the forum in order to post messages to the forum.

On servers where PHP is run as a CGI binary, the Archive mode may not work past the main forum listings.
