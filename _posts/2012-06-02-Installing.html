سپاس از شما برای انتخاب مای بی بی برای نرم افزار انجمن ساز خود. شما فقط نیاز‌هست مراحل زیر را قدم به قدم برای کامل شدن انجمن جدیدتان انجام دهید. ما مراحل نصب را به مراحل کوچکتری تقسیم کرده‌ایم که می‌توانید در سمت چپ آنها را ببینید. اگر برایتان مشکلی پیش آمد یا سوالی داشتید، می‌توانید در انجمن‌های پشتیبانی سوال خود را پرسیده و پاسخ بگیرید.

  • مای‌بی‌بی دارای ویژگی‌هایی که از یک انجمن‌ساز انتظار می‌رود هست و همچنین برخی ویژگی‌هایش در دیگر انجمن‌سازها یافت نمی‌شود.
  • مشکلی با مای‌بی‌بی دارید؟ٔ پشتیبانی در انجمن‌های پشتیبانی مای‌بی‌بی برای هرکس که از مای‌بی‌بی استفاده می‌کند رایگان است، این پشتیبانی توسط تیم پشتیبانی و دیگر کاربران انجمن انجام می‌شود!
  • مای‌بی‌بی به راحتی قابل سفارشی‌سازی به سایتی که مورد نیاز شماست، هست.
  • بهتر از همه، آن هم به صورت رایگان! بدون هیچ تبلیغ اجباری‌، بدون هزینه و بدون هیچ اشتراک ایمیل اجباری!
  • شما به یک هاست برای مای‌بی‌بی نیاز خواهید داشت.
  • هاست شما به نیازمندی‌های ساده‌ای برای اجرای مای‌بی‌بی نیاز دارد.

ز وب سایت پشتیبانی مای بی بی دانلود کنید. بعد از دانلود فایل فشرده را از فشرده خارج کنید.در ویندویز شما باید روی فایل راست کلیک کرده و گزینه "Unzip" را انتخاب کنید.هم چنین شما می توانید از برنامه های دیگر چون WinRar برای خارج کردن از فشرده استفاده کنید. پس از فشرده خارج کردن محتویات فایل را به یک پوشه در همان پوشه که فایل های فشرده است، مانند یک پوشه به نام "MyBB".

بعد از خارج کردن از فشرده سازی شما نیاز دارید که فایل مای بی بی را در سرور خود آپلود کنید.

برای نصب مای بی بی در mySQL 4.0.x، شما نیاز دارید چند کار را قبل از نصب انجام دهید. این یک راه حل است نه یک جایگزین و البته نسخه mySQL 4.0 قدیمی ترین نسخه ای می باشد که مای بی بی بر روی آن نصب می شود.

در ./inc/db_mysql.php، کد زیر را حذف کنید: 

		if($success && $this->db_encoding)
		{
			$this->query("SET NAMES '{$this->db_encoding}'");
			if($write_success && count($this->connections) > 1)
			{
				$this->write_query("SET NAMES '{$this->db_encoding}'");
			}
		}
		return $success;

و همچنین: 

		return " CHARACTER SET {$this->db_encoding} COLLATE {$collation}";

این کار نصب مای‌بی‌بی را آزاد می‌کند. اگر هر مشکلی بود، لطفا به تیم ما در انجمن پشتیبانی گزارش دهید.

نکته: این تغییرات برای هاست‌هایی با MySQL 5.0 یا بیشتر نیاز نیست.

Aries-Belgium و Dylan M. متد‌های نصب را به‌گونه‌ای توسعه دادند تا با استفاده از shell و چند خط کد نیز بتوان مای‌بی‌بی را نصب کرد.

  • دسترسی - خط‌های فرمان نیاز به دسترسی به سرور دارند
  • خط‌های فرمان از wget, aria2 (version >= 1.9.3), curl, or lynx استفاده می‌کنند، بستگی دارد که شما کدام را انتخاب می‌کنید. همچنین سرور شما باید روی *NIX (Linux, UNIX, یا Mac) باشد.
  • unzip, mv, rm, and chmod - اینها نیازمندی‌های خط‌های فرمان هستند و باید در سرور شما باشند. 
wget --content-disposition http://www.mybb.com/download/latest -O mybb.zip && unzip mybb.zip "Upload/*" && mv Upload/* . && rm -Rf Upload mybb.zip && mv inc/config.default.php inc/config.php && chmod -R 0777 cache uploads inc/settings.php inc/config.php
aria2c http://www.mybb.com/download/latest -o mybb.zip && unzip mybb.zip "Upload/*" && mv Upload/* . && rm -Rf Upload mybb.zip && mv inc/config.default.php inc/config.php && chmod -R 0777 cache uploads inc/settings.php inc/config.php
curl http://www.mybb.com/download/latest -o mybb.zip && unzip mybb.zip "Upload/*" && mv Upload/* . && rm -Rf Upload mybb.zip && mv inc/config.default.php inc/config.php && chmod -R 0777 cache uploads inc/settings.php inc/config.php
lynx -crawl -dump http://www.mybb.com/download/latest > mybb.zip && unzip mybb.zip "Upload/*" && mv Upload/* . && rm -Rf Upload mybb.zip && mv inc/config.default.php inc/config.php && chmod -R 0777 cache uploads inc/settings.php inc/config.php
